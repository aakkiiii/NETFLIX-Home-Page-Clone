

# 🎬 Netflix Landing Page Clone

A responsive Netflix landing page clone built using **HTML**, **CSS**, and **JavaScript**. This project recreates the look and feel of Netflix's homepage with a modern user interface, responsive design, and interactive elements. It demonstrates strong front-end development skills by focusing on layout design, styling, responsiveness, and user experience.

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Netflix-inspired homepage layout
- Hero section with background image and call-to-action
- Navigation bar with logo and buttons
- Multiple content sections highlighting features
- Interactive hover effects and smooth transitions
- Footer with useful links
- Clean and modern UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
